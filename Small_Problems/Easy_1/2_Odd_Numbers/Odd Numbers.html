Odd Numbers

CategoryEasy 1
CourseJS101
Exercise Number1.2
StatusCurrent

Problem Statement

Log all odd numbers for 1 to 99, inclusive, to the console. Log all numbers on separate lines.

Further Exploration

Repeat this exercise with a technique different from the one that you used, and different from the one provided. Also consider adding a way for the user to specify the limits of the odd numbers logged to the console.

Understanding the Problem

INPUTS

start: number integer range: -inf to inf

end: number integer range: -inf to inf

OUTPUTS

none

CLARIFYING QUESTIONS

What if user enters invalid inputs? (float, string, etc) assume valid inputs

What if user enters start and end backwards? do not print, raise no error

RULES

EXPLICIT:

  1. user enters start and end arguments
  1. print odd numbers from between, and including, start and 99 to the console

IMPLICIT:

  1. print those numbers where, when divided by two, yield no remainder

IDEAS

N/A

Examples / Test Cases

Rule Set Derived Cases

oddNumbers(1, 99) ⇒ (prints odds 1 to 99)

oddNumbers(-10, 10) ⇒ (prints odds -9 to 9)

Invalid Input Derived Cases

oddNumbers(10, -10) ⇒ (no lines printed)

Edge Derived Cases

None (no input)

Data Structures

INPUTS

None

OUTPUTS

None

Algorithm

1 - set index value equal to start

2 - loop while index value is not greater than end value, for each iteration

2.1 - if index is not evenly divisible by 2, log index to the console

2.2 - increment index
