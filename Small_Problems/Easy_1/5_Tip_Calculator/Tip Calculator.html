Tip Calculator

CategoryEasy 1
CourseJS101
Exercise Number1.5
StatusCurrent

Problem Statement

Create a simple tip calculator. The program should prompt for a bill amount and tip rate. The program must compute the tip, and then log both the tip and the total amount of the bill to the console. You can ignore input validation and assume that the user will enter numbers.

Example

What is the bill? 200
What is the tip percentage? 15

The tip is $30.00
The total is $230.00

Understanding the Problem

INPUTS

bill: number range: 0 to inf

tipPercent: number range: 0 to inf

OUTPUTS

tipTotal: number

billTotal: number

CLARIFYING QUESTIONS

Can the tip percentage be a float value? - yes

Should fractional cents be rounded up or down? - round up

RULES

EXPLICIT:

  1. store user prompted values for bill amount and tip percent
  1. display tip amount and total bill amount based on input values

IMPLICIT:

  1. calculated tip amount and bill amount should display dollar and cents values in output
  1. user is able to enter float values for bill (cents) and percentage

IDEAS

N/A

Examples / Test Cases

Rule Set Derived Cases

  1. Input(200, 15) ⇒ tip: 30.00, bill: 230.00
  1. Input(15, 10) ⇒ tip: 1.50, bill: 16.50
  1. Input(99, 200) ⇒ tip: 198.00, bill: 297.00
  1. Input(200.5, 10.5) ⇒ tip: 2.11, bill: 202.61

Invalid Input Derived Cases

None

Edge Derived Cases

  1. Input(0, 20) ⇒ 0, 0
  1. Input(0, 0) ⇒ 0, 0
  1. Input(20, 0) ⇒ 0, 20

Data Structures

INPUTS

numbers

OUTPUTS

numbers

Algorithm

1 - prompt user for tip and bill values, set to tipPercent and billAmt

2 - calculate tipAmt as tipPercent * billAmt and round to nearest whole cent

3 - calculate billTotal as tipAmt plus billAmt
